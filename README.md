## Система оповещения о диких животных у вашего хозяйства

### Проблема

Не у всех фермеров есть возможность обзавестить полноценной охранной системной. Есть множество примеров когда дикие животные часто выходят из своих постоянных мест обитания в поисках пищи. Системой, на базе данного решения, смогут обзавестить множество фермеров, с разным уровнем своих владений.

---

### Решение

Разработка **интеллектуальной системы мониторинга и раннего оповещения**, основанной на **компьютерном зрении и детекции крупных животных** в приграничных зонах ООПТ.

#### Технические компоненты

| Элемент Системы | **Входные Данные / Триггер** | **Функция / Реализация IT-Решения** |
| :--- | :--- | :--- |
| **Детекция Животных** | Постоянно работающие **фотоловушки** (с передачей по 4G/спутниковой связи) вдоль границ ООПТ. | **Модель машинного обучения (SAM3)**, обученная на изображениях местных животных (медведь, волк, рысь, лиса, кабан). Задача: **Детекция и классификация объектов** (object detection). |
| **Визуализация и Анализ** | Фотоловушки | Результаты детекции: **название животного, количество особей, координаты, время, уверенность**. |
| **Автоматическое Оповещение** | Фотоловушки | Детекция животного в пределах **"буферной зоны"** (например, 5 км от ближайшего населенного пункта). **Автоматический триггер** (Telegram/Email-бот) для немедленного оповещения егерей, местных властей и фермеров. |
